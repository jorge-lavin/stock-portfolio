Este proyecto consiste en una base de datos de datos historicos de empresas.

Se proveen las siguientes entidades:

Stock
-------
Representa una empresa, en un país y un sector.

TODO ¿Cómo vamos a hacer con empresas que cotizan en varios mercados?

Country
-------
Representa un país. Se utiliza el estándar ISO 3166-1 https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2

Sector
------
Representa un sector. Se utiliza la ontología y clasificación de Morningstar. 

Dividend
--------
Representa el dividendo de una empresa, en una fecha dada, devengado en una cierta divisa.

Currency
--------
Modela una divisa. Se utiliza el estándar ISO 4217 https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2
